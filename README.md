# Case Management

App To Manage Cases — built on Frappe Framework.

## Compatibility

- **Frappe/ERPNext v15** ✅

## Installation

```bash
bench get-app https://github.com/balaji-001-gif/case_management.git
bench --site your-site install-app case_management
bench --site your-site migrate
bench build
```

## Modules

- **Case Management** — GBV, CP, and General case management with assessments, action plans, and closure forms
- **CPBNs** — Community-based Protection Networks with monitoring tools, service mapping, and referrals
